Get sum of an array using recursion

My understanding of the code:
If the length of the array is 0, the function would return 0
Else, the sum of the array is the result of adding the first element to the remainder of the array until there's nothing left in the array which is 0
5 + sum([9, 6, 8, 4, 6])
5 + 9 + sum([6, 8, 4, 6])
5 + 9 + 6 + sum([8, 4, 6])
5 + 9 + 6 + 8 + sum([4, 6])
5 + 9 + 6 + 8 + 4 + sum([6])
5 + 9 + 6 + 8 + 4 + 6 + sum([])
